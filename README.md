# 💼 Portfolio Website

This is a responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript**.
It showcases sections like About, Skills, Projects, and Contact information,
all styled with modern design principles and optimized for both desktop and mobile views.

> 🚧 This project includes only the **frontend part** and does not contain any backend functionality.

---

## ✨ Features

- ⚡ Responsive navigation bar with hamburger menu for mobile
- 🎯 Smooth scrolling to different sections
- 📱 Mobile-first design with media queries
- 🧑 About Me section with styled layout
- 💻 Projects section to display your work
- 📋 Contact and Social Links (static)

---

## 🛠 Technologies Used

- **HTML5**
- **CSS3**
- **Vanilla JavaScript (ES6)**

---

## 📱 Responsive Design

- Fully mobile-friendly
- Hamburger menu toggle implemented in JavaScript
- Media queries ensure layout adapts on various screen sizes
